# 📊 Project Controls Power BI Dashboard

An interactive **Project Controls Dashboard** built in **Microsoft Power BI** to monitor project performance through cost, schedule, revenue, margin, and resource utilization KPIs. The dashboard provides stakeholders with real-time insights into project health, enabling informed decision-making and proactive project control.

---

## 📌 Project Overview

This dashboard helps project managers and stakeholders monitor project performance by comparing **planned vs. actual** metrics. It combines financial, operational, and scheduling data into a single interactive report for effective project monitoring.

---

## 🎯 Objectives

- Monitor project financial performance
- Track project progress and completion status
- Compare planned vs. actual performance
- Analyze project profitability
- Identify project risks and delays
- Support data-driven decision-making

## 💡 Key Insights

- Revenue achieved **75%** of the planned target.
- Project costs remain within budget at **81%** of the planned amount.
- Current profit margin is **43%** of the planned target.
- Most project tasks are progressing **On Track**, while a small number require attention due to delays or risks.
- Billable hours represent approximately **88%** of total recorded hours, indicating efficient resource utilization.
